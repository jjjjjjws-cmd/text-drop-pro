# 📋 텍스트 드롭 프로그램 PRO

클릭으로 복사, 드래그로 붙여넣기가 가능한 텍스트 관리 도구 + 엑셀 뷰어

## 🎯 주요 기능

### 📝 텍스트 관리
- ✅ **클릭 복사**: 텍스트 카드 클릭 시 즉시 복사
- ✅ **드래그 드롭**: 다른 사이트/앱에 드래그하여 붙여넣기
- ✅ **카테고리 관리**: 업무, 개인, 고객사 등으로 분류
- ✅ **수정 기능**: 저장된 텍스트 언제든지 수정 가능
- ✅ **로컬 저장**: 브라우저에 데이터 저장 (개인별 독립적)

### 📊 엑셀 뷰어
- ✅ **파일 업로드**: .xlsx, .xls, .csv 지원
- ✅ **멀티 시트**: 여러 시트 탭 전환
- ✅ **셀 복사/드래그**: 클릭 복사 + 드래그 드롭
- ✅ **검색 기능**: 실시간 검색 + 하이라이트
- ✅ **컬럼 필터**: 특정 컬럼만 검색 가능
- ✅ **필터 정보**: 표시 행 수 안내

## 🚀 사용 방법

### 온라인 접속
👉 **[여기를 클릭하여 바로 사용](https://YOUR-USERNAME.github.io/text-drop-pro/)**

### 텍스트 저장
1. 카테고리 입력 (선택)
2. 라벨 입력 (필수)
3. 내용 입력 (필수)
4. 저장 버튼 클릭

### 텍스트 사용
- **클릭**: 즉시 복사
- **드래그**: 다른 곳에 드롭

### 엑셀 뷰어
1. "📊 엑셀 뷰어" 탭 클릭
2. 엑셀 파일 업로드
3. 검색창에 검색어 입력
4. 원하는 셀 클릭 또는 드래그

## 💾 데이터 저장

모든 데이터는 **브라우저의 localStorage**에 저장됩니다.
- ✅ 각 사용자별 독립적 저장
- ✅ 브라우저 닫아도 데이터 유지
- ✅ 다른 사용자는 볼 수 없음

## 🔒 개인정보 보호

- ✅ 서버에 데이터 전송 안 함
- ✅ 모든 데이터는 사용자 PC에만 저장
- ✅ 100% 프론트엔드 동작

## 📱 지원 환경

- ✅ Chrome, Edge, Firefox, Safari
- ✅ Windows, Mac, Linux
- ✅ 모바일 브라우저
